## **Prerequisites**

- Go version 1.25 or higher.
- Otim developer keys from the [getting your developer keys](/sdk/getting-your-developer-keys) step.

## **Install**

The Otim Go SDK is available on GitHub and can be installed using `go get`:

```bash
go get github.com/otimlabs/otim-go-sdk
```

### **Go Module Requirements**

Ensure your project is initialized as a Go module:

```bash
go mod init your-project-name
```

The SDK requires Go 1.25+ and uses standard Go modules for dependency management.
