| Term              | Description                                                                                                                                                                   |
| :---------------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Action Contract   | An Otim smart contract that implements a specific feature for the Otim protocol.                                                                                              |
| Delegate Contract | A general term for a smart contract designed for EOAs to delegate to via EIP-7702 transaction. See our appendix entry on [EIP-7702](/docs/resources/appendix/eip-7702) for more. |
| Instruction       | A primitive data type for the Otim protocol that defines how an Action contract will be executed on behalf of a given user.                                                   |
