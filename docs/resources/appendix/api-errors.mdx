This document contains all user-addressable error codes (1xxx) that can be returned by the API. These errors are categorized by their retry behavior and error type.

<Tabs>
  <Tab title="Terminal">
    <Tabs>
      <Tab title="Contract-Level">
        | Code | Description |
        | :--- | :--- |
        | 1201 | User account is not delegated to the OtimDelegate contract |
        | 1202 | Signature verification failed for the provided instruction |
        | 1203 | Action cannot be executed due to contract-level restrictions |
        | 1205 | Instruction has already been deactivated and cannot be executed |
      </Tab>
      
      <Tab title="Transfer Action">
        | Code | Description |
        | :--- | :--- |
        | 1301 | Invalid or malformed arguments provided for transfer action |
        | 1302 | SafeERC20 transfer operation failed due to token contract issues |
      </Tab>
      
      <Tab title="Transfer ERC20 Action">
        | Code | Description |
        | :--- | :--- |
        | 1305 | Invalid or malformed arguments provided for ERC20 transfer action |
        | 1306 | SafeERC20 transfer operation failed due to token contract issues |
      </Tab>
      
      <Tab title="Refuel Action">
        | Code | Description |
        | :--- | :--- |
        | 1309 | Invalid or malformed arguments provided for refuel action |
        | 1310 | SafeERC20 transfer operation failed due to token contract issues |
      </Tab>
      
      <Tab title="Refuel ERC20 Action">
        | Code | Description |
        | :--- | :--- |
        | 1314 | Invalid or malformed arguments provided for ERC20 refuel action |
        | 1315 | SafeERC20 transfer operation failed due to token contract issues |
      </Tab>
      
      <Tab title="UniswapV3 Exact Input Action">
        | Code | Description |
        | :--- | :--- |
        | 1319 | Invalid or malformed arguments provided for UniswapV3 swap action |
        | 1320 | UniswapV3 pool does not exist for the specified token pair and fee tier |
        | 1321 | Unexpected invalid tick data encountered during pool validation |
        | 1322 | SafeERC20 transfer operation failed due to token contract issues |
      </Tab>
      
      <Tab title="Sweep Deposit Account Action">
        | Code | Description |
