Below is the first batch of Actions currently available to users, but many more Actions are in development!

| Action    | Description                                                                                                        |
| :-------- | :------------------------------------------------------------------------------------------------------------------|
| Transfer  | Send tokens to a target account at scheduled intervals.                                                            |                                                             |
| Refuel    | Track the balance of a target account and send tokens if the target's token balance falls below a given threshold. |
| Swap      | Swap tokens using Uniswap V3 pools on a recurring basis.                                                           |
| Sweep     | Any funds sent to a pre-defined address gets swept to any chosen destination                                       |
| Call      | Execute an arbitrary function call at scheduled intervals.                                                         |
| MultiCall | Execute a batch of arbitrary function calls at scheduled intervals.                                                |

Find all of the above Actions' deployed contract addresses [here](/docs/developers/deployed-addresses).

Reference the 712 hash ABI's for each Action [here](/docs/resources/appendix/action-hash-abis).
